## Full Description (max 4000 chars)
GAMA Mobile brings the power of GAMA — a leading open-source platform for Agent-Based Modeling (ABM) — to your Android device. Run complex spatial simulation models anywhere, anytime, without needing a desktop or remote server.

**What's Inside:**
• Full GAMA headless runtime with Java 25 (Temurin JRE)
• Complete Debian Linux environment via PRoot container
• Interactive VNC viewer with touch and keyboard support
• Real-time simulation monitoring with live logs
• One-tap startup — no configuration required

**Key Features:**
• **Dashboard** — Monitor your simulation's status, progress, and connection state at a glance
• **Live VNC** — Interact with GAMA's full desktop interface directly on your phone or tablet with optimized touch controls
• **Log Viewer** — Follow simulation output in real-time with color-coded log levels and auto-scroll
• **Fully Offline** — Everything runs locally on your device. No internet connection needed after install.

**Built for Researchers & Modelers:**
Whether you're a researcher, student, or policy maker, GAMA Mobile lets you prototype, test, and demonstrate agent-based models on the go. Load your existing GAMA models and run them anywhere.

**Technical Details:**
• ARM64 architecture (Android 8.0+)
• Debian Bookworm rootfs with Python 3 + websockets
• Xvfb virtual display with X11vnc
• React Native UI with responsive landscape layout

GAMA is developed by IRD/UMI 209 UMMISCO and partners. Learn more at gama-platform.org.
